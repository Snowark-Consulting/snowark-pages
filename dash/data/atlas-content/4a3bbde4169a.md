# Registry — Not Forgotten Systems

> **Map, not a library.** Read this first every session. Follow the pointers; don't scan everything.

See [[SOPs]] (Agent/SOPs.md) for recurring procedures, [[Infrastructure]] (Agent/Infrastructure.md) for what's running, and [[MEMORY]] (memory/MEMORY.md) for cross-session facts.

## How this works

1. **This file** is the master map (~40 lines). Read it every session.
2. Each line points to a **domain index** with a one-line hook describing what's there.
3. Domain indexes hold granular pointers to detail cards, strategy docs, reports, etc.
4. Detail cards point to the **full source** (email, drive, system, file path).

Three hops max: Registry → Domain Index → Detail Card → Source.
