# Not Forgotten Systems — Services

> Part of [[Business Overview]] (knowledge/business.md). See also [[Products & Services]] (knowledge/products.md) for the SaaS platform.

## Core Service
Automated quote follow-up for service businesses.

## Features
- Email follow-up sequences (automated)
- SMS follow-up sequences (automated)
- Quote tracking and response monitoring
- Business owner notifications
- Simple CC/BCC integration — no API setup required

## Onboarding (included in $99 setup)
- Account setup
- Workflow configuration
- Email connection
- SMS configuration
- Onboarding walkthrough

## Use Cases
See [[Customers]] (knowledge/customers.md) for target profile and case studies.
- Trade businesses sending quotes and losing track of follow-up
- Home service companies wanting automated touchpoints
- Professional service firms managing multiple quotes
- Any business where quote follow-up is manual or forgotten
